Features

Everything you need

Powerful features to keep your team informed about every Shopify platform change.

Real-time Notifications

Get instant alerts when Shopify publishes new changelog entries. Never be caught off-guard by breaking changes.

Smart Filtering

Filter by categories, keywords, or exclusions. Only get notified about the updates that matter to your team.

Priority Detection

Automatically flags critical updates like breaking changes, security patches, and deprecations.

Express Middleware

Seamlessly integrate into your Node.js applications with our Express middleware support.

Docker Ready

Deploy anywhere with our Docker support. Run as a container with persistent state tracking.

Flexible Config

Use JSON or JavaScript config files. Override settings via CLI flags or environment variables.

Usage

Simple configuration

Get started with a simple config file or use CLI flags for quick setup.

// shopify-monitor.config.js
module.exports = {
  webhooks: {
    slack: ['https://hooks.slack.com/...'],
    teams: ['https://outlook.office.com/...'],
  },
  checkInterval: 15,
  filters: {
    categories: ['API', 'Apps'],
    keywords: ['breaking', 'deprecated'],
  }
};
